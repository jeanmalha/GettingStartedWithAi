# Getting Started with AI

A series of Jupyter notebooks for developers who want to build a solid intuition for how AI works — from transistors to transformers.

## Notebooks

| # | Notebook | Topics |
|---|---|---|
| 01 | [What is AI?](01-what-is-ai.ipynb) | Definition, transistors, logic gates, coded logic, symbolic AI, deep learning, perceptrons |

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
jupyter notebook
```

For Claude API notebooks, set your API key:

```bash
export ANTHROPIC_API_KEY=your_key_here
```
